¿ Qué es HTML 5 ?

HTML es el acrónimo de HyperText Markup Language (Lenguaje de Marcado de Hipertexto). HTML es un lenguaje de etiquetas,que indica al navegador como tiene que mostrar el contenido.HTML emplea un conjunto de elementos predefinidos que permitenidentificar los distintos tipos de elementos. Estos elementoscontienen una o más etiquetas que contienen o expresan elcontenido. Estas etiquetas suelen ir encapsuladas entre los símbolos(<>), y las etiquetas de cierre (que indican el final de un determinado contenido) están precedidas por una barra /.

ETIQUETAS Y ATRIBUTOS HTML


ETIQUETAS


Las etiquetas HTML son fragmentos de texto rodeados por corchetes angulares , que tienen funciones y usos específicos y se utilizan para escribir código HTML.HTML define 91 etiquetas que los diseñadores pueden utilizarpara marcar los diferentes elementos que componen una página.s De todas las etiquetas disponibles, las siguientes se consideran obsoletas y no se pueden utilizar: applet,basefont,center,dir,font,insindex,menu,s,strike,u.A pesar de que se trata de un número de etiquetas muy grande, no es suficiente para crear páginas complejas. Algunos elementos como las imágenes y los enlaces requieren cierta información adicional para estar completamente definidos

ATRIBUTOS


No todos los atributos se pueden utilizar en todas las etiquetas. Por ello, cada etiqueta define su propia lista de atributos disponibles. Además, cada atributo también indica el tipo de valor que se le puede asignar. Si el valor de un atributo no es válido, el navegador ignora ese atributo. Aunque cada una de las etiquetas HTML define sus propios atributos, algunos de los atributos son comunes a muchas o casi todas las etiquetas. De esta forma, es habitual explicar por separado los atributos comunes de las etiquetas para no tener que volver a hacerlo cada vez que se explica una nueva etiqueta.

ATRIBUTOS BÁSICOS

La mayoría de páginas web actuales utilizan los atributos id y class de forma masiva. Sin embargo, estos atributos sólo son realmente útiles cuando se trabaja con CSS y con Javascript. Respecto al valor de los atributos id y class, sólo pueden contener guiones medios (-) , guiones bajos (_), letras y/o números, pero no pueden empezar por números. Además, los navegadores distinguen mayúsculas de minúsculas y no se recomienda utilizar letras como y acentos, ya que no es seguro que funcionen correctamente en todas slas versiones de todos los navegadores.
